Kyrgyzstan Deposit Calculator

Простой и полезный финансовый калькулятор, который помогает людям считать прибыль с депозита, учитывая то, как инфляция сьест их доходы

Ключевые функции
- учет инфляции: рассчитывает потерю покупательной способности заданной суммы
- налоговые ветки: для Резидентов КР и Нерезидентов КР
- реальная ставка: Показывает процент, который вы получаете по факту


Технологии:
* Python 3.x
* Tkinter (GUI)


как запустить:
1. убедитесь, что у вас установлен Python.
2. скачайте файл `main.py`.
3. запустите его командой: `python main.py`
